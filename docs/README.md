# 99.95

A Chrome Extension that helps students stay on top of their schedule with a class countdown timer, integrated datasheets, Desmos, and light/dark mode support.

[**Install from Chrome Web Store**](https://chromewebstore.google.com/detail/9995/cjbldnpcbmbpofalefjjnfapdgknjibd)

## Features

- **Class Schedule Countdown** — see how much time is left in your current period at a glance
- **Integrated Datasheets** — quick access to reference sheets without leaving your browser
- **Desmos Integration** — built-in graphing calculator access
- **Light / Dark Mode** — toggle between themes to match your preference

## Tech Stack

- JavaScript · CSS · HTML
- Chrome Extensions API (Manifest V3)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Art4v/99.95-chrome-extension.git
   ```
2. Open `chrome://extensions` in your browser
3. Enable **Developer mode**
4. Click **Load unpacked** and select the cloned folder
